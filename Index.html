Comandos básicos de Git
Comando Descripción
git init Inicializa un nuevo repositorio en la carpeta actual.
git clone <url> Descarga un repositorio remoto a tu computadora.
git status Muestra el estado de los archivos y la rama actual.
git add <archivo> Agrega un archivo al área de preparación (staging).
git commit -m "mensaje" Guarda los cambios en el historial con un mensaje.
